📦 Order Management System
📌 Description

This project is an Order Management System that allows users to manage clients, products, and orders efficiently.

It is designed to simulate a real business workflow where customers place orders, products are managed, and transactions are recorded.

The system provides a simple and practical way to organize data and automate basic business operations.

🚀 Features

👤 Client Management

Register new clients

View and store client information

📦 Product Management

Add new products

Store product details such as name, price, and stock

🧾 Order Management

Create orders linked to clients

Add multiple products to an order

Calculate totals automatically

💾 Data handling using structured storage (lists/files)

🛠️ Technologies Used

Python 🐍

JSON / File handling

Standard libraries

⚙️ Installation

Clone the repository:

Navigate to the project folder:

cd Proyecto-gestion-pedidos-clientes

Run the program:

python main.py
